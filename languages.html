Languages supported

From version 2.1.0 onwards, LiClipse also supports TextMate bundles by adding a '.zip' file renamed to '.tmbundle' to a tracked folder, which means it can currently support a huge number of languages.

See TextMate Bundles Integration for more details.

Out of the box it provides support for:

  • C/C++
  • CSS
  • CoffeScript
  • Dart
  • Django Templates
  • Go (new on 2.0)
  • HTML
  • Java
  • JavaScript
  • Jinja2 (new on 2.0)
  • Julia
  • Kivy Language
  • Linux Shell files (new on 2.2.0)
  • Mako Templates
  • Markdown (new on 2.1.0)
  • Nim (new on 2.0)
  • PERL (new on 2.2.0)
  • PHP (new on 2.1.0)
  • Python
  • RAML (new on 2.4.0)
  • RST (ReStructured Text)
  • Ruby (new on 2.1.0)
  • SCSS
  • StringTemplate
  • Swift (new on 2.2.0)
  • Windows bat files (new on 2.2.0)
  • XML
  • YAML
