Languages supported

LiClipse supports many languages out of the box:

  • C/C++
  • CoffeScript
  • CSS
  • Dart
  • Django Templates
  • Go (new on 2.0)
  • HTML
  • Java
  • JavaScript
  • Jinja2 (new on 2.0)
  • Julia
  • Kivy Language
  • Ruby (new on 2.1.0)
  • Mako Templates
  • Markdown (new on 2.1.0)
  • Nim (new on 2.0)
  • Python
  • PHP (new on 2.1.0)
  • RST (ReStructured Text)
  • SCSS
  • StringTemplate
  • XML
  • YAML

Additionally, since LiClipse 2.1.0, TextMate bundles are also supported, which means that lots of other languages can also be easily added to LiClipse.

See TextMate Bundles Integration for more details.
