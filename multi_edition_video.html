The video below shows improvements on the multi-edition and code-completion for the LiClipse HTML editor.

Multi-edition below is activated with:

  • Ctrl+K (forward search) or Ctrl+Shift+K (backward search)
  • Ctrl + Alt + double-click a word to add it to the multi-edition or Ctrl + Alt + Click to select the mouse-clicked position.
  • Ctrl + Alt + mouse drag can be used to select a region (if the selection starts and ends in the end of the line it will select only the line ends, otherwise it'll select the whole lines).

A selected region can be unlinked with Shift + Alt + K
